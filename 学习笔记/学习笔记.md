### 1.0 稀疏数组 sparseArray
五子棋存储
![五子棋](.\五子棋.jpg)
围棋棋盘由纵横各19条等距离、垂直交叉的平行线构成，形成361个交叉点。

使用二位数组记录棋盘 => 稀疏数组 => 保存为磁盘文件

稀疏数组 sparseArray的结构

```
第0行： row，col， value
第1行：  行，  列， 值
...................
第n行：  行，  列， 值
```

二维数组 转 稀疏数组 的思路：
 1. 遍历原始的二维数组，得到有效数据的个数sum
 2. 根据sum创建稀疏数组sparseArray int[sum+1][3]
 3. 将二位数组的有效数据存入到 稀疏数组中
